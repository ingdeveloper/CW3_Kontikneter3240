Neustarts aller Clients auslösen

Bei Betätigung des Buttons wird ein Neustart aller verbundenen Clients ausgeführt!
Nach einem Klick wird der Button gelb angezeigt, dann wird nach ca. 3 Sekunden ein Reload der Seite gemacht. Die Seiten werden mit der Javascript-Function location.reload(true); , also refresh der Server-Seiten, geladen.
Durchführung ist in der shell.ts programmiert.
https://wiki.selfhtml.org/wiki/JavaScript/Location/reload: Optional kann dieser Funktion der Parameter true übergeben werden, um ein Neuladen der Seite vom Server zu erzwingen. Ohne diesen Parameter wird die Seite aus dem Zwischenspeicher des Browsers (Cache) neu geladen.
